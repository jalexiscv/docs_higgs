###################
Ayudante de números
###################

El archivo Number Helper contiene funciones que le ayudan a trabajar con
datos numéricos teniendo en cuenta la configuración regional.

.. contents::
    :local:
    :depth: 2

======================

.. php:function:: number_to_amount($num[, $precision = 1[, $locale = null])

    :param mixed $num: Número a formatear
    :param int $precision: Precisión de punto flotante
    :param string $locale: la configuración regional que se utilizará para formatear
    :devuelve: una versión legible por humanos de la cadena, o false si el valor proporcionado no es numérico
    :rtype: cadena

    Convierte un número a una versión legible por humanos, como **123,4 billones**
    para números hasta los cuatrillones. Ejemplos:

    Un segundo parámetro opcional le permite establecer la precisión del resultado:

    .. literalinclude:: number_helper/006.php

    Un tercer parámetro opcional permite especificar la configuración regional:

    .. literalinclude:: number_helper/007.php

    :param entero $fracción: Número de dígitos de fracción después del punto decimal
    :devuelve: El número como moneda apropiada para la configuración regional

    Si no especifica una configuración regional, se utiliza la configuración regional de Solicitud.

.. php:function:: number_to_roman($num)

    :param int|string $num: El número que desea convertir
    :returns: el número romano convertido a partir del parámetro dado
    :rtype: cadena|nulo

    Convierte un número a romano:

    .. literalinclude:: number_helper/009.php

    Esta función solo maneja números en el rango del 1 al 3999.
    Devolverá "nulo" para cualquier valor fuera de ese rango.
